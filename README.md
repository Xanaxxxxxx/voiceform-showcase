Voiceform is an on-device voice mirror for real-time pitch, resonance, thinness, and voice exploration.

A calm, supportive iPhone experience for people shaping a voice that feels like their own.

## Overview

- on-device voice practice experience on iPhone
- real-time feedback across pitch, resonance, thinness, and articulation surfaces
- local-first design for private voice exploration
- no source code or internal implementation details in this public showcase

## Product Lens

Voiceform is intended to feel supportive rather than judgmental, precise without becoming clinical, and expressive without losing technical credibility.

The core loop is simple: speak, see live feedback, adjust, and try again when it feels useful.
