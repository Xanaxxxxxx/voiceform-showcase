# Voiceform Support

For support, beta feedback, or privacy questions, contact:

`gliding_corers0h@icloud.com`

When reporting an issue, please include:

- device model
- iOS version
- app version and build number
- what you were doing before the issue happened
- whether microphone permission was enabled

For TestFlight builds, you can also use TestFlight's Send Feedback button to include logs and screenshots through Apple's feedback system.
